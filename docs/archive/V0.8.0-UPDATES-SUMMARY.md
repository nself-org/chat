## Quick Reference Updates

### Installation Commands

**Web:**
```bash
git clone https://github.com/acamarata/nself-chat.git
cd nself-chat
pnpm install
pnpm dev
```

**iOS:**
```bash
pnpm build
cd platforms/capacitor
pnpm run sync:ios
pnpm run open:ios
```

**Android:**
```bash
pnpm build
cd platforms/capacitor
pnpm run sync:android
pnpm run open:android
```

**Desktop:**
```bash
pnpm electron:dev  # Development
pnpm electron:build  # Production
```
