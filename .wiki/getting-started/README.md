## ðŸ’¡ Quick Tips

### Development Mode

nself-chat includes 8 test users for development:

- `owner@nself.org` - Full permissions
- `admin@nself.org` - User/channel management
- `moderator@nself.org` - Content moderation
- `member@nself.org` - Standard user
- `guest@nself.org` - Limited read-only

**Password for all:** `password123`

### Zero Backend Configuration

The nself CLI handles all backend infrastructure:

- PostgreSQL database
- Hasura GraphQL engine
- Nhost authentication
- MinIO storage
- MeiliSearch
- Redis

Just run `nself start` and everything is ready!
